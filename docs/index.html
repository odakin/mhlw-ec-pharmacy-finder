    よくある質問

    処方箋なしで買えますか？

    はい。要指導医薬品として、全国10,000件以上の薬局で処方箋なしで購入できます。薬剤師による対面での確認があります。在庫や勤務状況は変動するため、来局前の電話確認が推奨されています。

    薬局と医療機関、どちらに行けばいいですか？

    急ぎの場合はまず薬局を検索してください。近くに見つからない場合や医師に相談したい場合は「常時在庫ありの医療機関も表示」をONにすると、対面診療で処方を受けられる医療機関（2,900件以上）も検索できます。

    事前に電話は必要ですか？

    薬局によります。「事前連絡『要』を除く」フィルターで、予約なしで来局できる薬局を絞り込めます。

    女性薬剤師や個室のある薬局を探せますか？

    はい。「女性薬剤師がいる」「個室あり」のフィルターでそれぞれ絞り込めます。ただし勤務状況は変動するため、来局前にご確認ください。
