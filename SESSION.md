## プロジェクト概要

厚労省の緊急避妊薬対応 薬局一覧（10,128件）と医療機関一覧（3,107件）を検索しやすくした非公式ツール。静的Webサイト（バニラJS + GitHub Pages）。

## 現在の状態（2026-03-23）

**全機能実装済み・運用中。** 未完了の開発タスクなし。

### データ

- 薬局: 9,951件（厚労省公表10,128件のうち住所あり。177件は統合済み空レコードで除外）
- 医療機関: 2,964件（常時在庫ありのみ。全3,107件中）
- ジオコーディング: 薬局 100%、医療機関 98.3%（東大CSIS）
- 営業時間パーサー: カバー率 97.1%（6,933種のフォーマット揺れ対応）
- データ更新: GitHub Actions で日次自動実行

### 件数表示の2層構造

- ハイライトカード: `meta.totalPublished`（10,128件 = 厚労省公表件数）
- ステータスバー: `DATA.length`（9,951件 = 検索可能件数）
- title/meta: 「全国10,000件以上」（概数表現、影響なし）
