## 直近の変更（2026-03-23）

### 医療機関パーサー改善（3点）

1. **obgyn URL断片混入修正**: PDF列溢れで obgyn に混入した URL 断片を `normalize_obgyn()` で除去（10件修正）
2. **stock 正規化**: `ほぼ有` `少々有` `0 有` 等のエッジケースを `normalize_stock()` で正規化（4件修正）
3. **営業時間パーサー拡張**: 医療機関データ対応のため normalizeHoursText + parseHours を強化
   - AM/PM → 24h変換、午前/午後ハンドリング
   - `Xを除く` パターンの汎用化（祝のみ → 全曜日）
   - 常時/年中/24時間+接尾辞の処理
   - 括弧曜日リストの早期除去（`(月・火・水) 9:00` → `月・火・水:9:00`）
   - 受付時間/毎週プレフィックス除去
   - 休診/対応不可パターンの拡張
   - 注記セグメントのスキップ（`return null` → 注記判定して `continue`）
   - 安全性修正: 24時間のグリーディマッチ修正（後続データ消失防止）、午後の時間範囲両端変換
   - parseTimeRange の $ アンカー除去: 時間範囲後の注記テキストを無視し、部分パースでの曜日欠落を50件修正
   - 括弧内除外注記の除去（`（除く水曜）` `(木は除く)` 等）、`Xを除くY` のプレフィックス除去
   - 括弧内休診注記の除去（`（水曜は休診）` `（木休診）` 等）追加
   - 結果: 薬局 97.1%→98.2%、医療機関 73.1%→88.3%
   - 医療機関カードでパース済み時間テーブルを表示（生テキストフォールバック付き）
